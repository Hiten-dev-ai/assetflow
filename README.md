# AssetFlow

Enterprise asset and shared-resource operations in one role-aware workspace.

## Core delivered

- Responsive operational dashboard with KPIs, overdue allocations, utilization and activity
- Employee-only account creation UX with protected role-promotion rules
- Shared relational schema for departments, employees, categories, assets, allocations, bookings, maintenance, notifications and activity logs
- Cloudflare D1-ready persistence declaration

## Local setup

```bash
npm install
npm run dev
```
